# Instrucciones para la ejecución

## Requisitos:
 - Versión del JDK 19
 - Version de chrome en máquina local 129...

## Pasos para ejecutar el proyecto

1. Clonar el repositorio en la máquina donde se va a ejecutar

2. Navega hasta la raíz del proyecto.

3. Para ejecutar las pruebas ejecuta el siguiente comando:
    Ejecución: mvn clean verify

4. Una vez ejecutadas las pruebas, los reportes generados los puedes consultar en el directorio:
 - Reporte de Serenity: target/site/serenity/index.html
